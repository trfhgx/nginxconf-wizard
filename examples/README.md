# Example Configurations

This directory contains example nginx configurations generated by the wizard for different use cases.

## Examples

- `static-site.conf` - Simple static website
- `react-spa.conf` - React Single Page Application with API
- `nextjs-app.conf` - Next.js SSR application
- `wordpress.conf` - WordPress site with PHP-FPM
- `microservices.conf` - Microservices API gateway
- `cdn-origin.conf` - Origin server behind Cloudflare

## Testing Examples

```bash
# Validate a configuration
nginxconf-wizard validate examples/static-site.conf

# Test with nginx
nginxconf-wizard test examples/static-site.conf

# Or use nginx directly
nginx -t -c examples/static-site.conf
```
